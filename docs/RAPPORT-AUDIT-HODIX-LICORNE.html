6. Roadmap P0 → P3

P0 — 0-30 jours

  • CinetPay production + webhook notify_url
  • Edge Function webhook confirmation paiements
  • Retrait wallet via payout Mobile Money réel
  • Tests E2E : auth, cotisation, recharge, retrait
  • Domaine hodix.app vérifié sur Resend

P1 — 1-3 mois

  • KYC : CNI + selfie + vérification tierce
  • Conformité COBAC/CEMAC
  • Retirer/unifier backend FastAPI legacy
  • Observabilité Sentry + alertes fraude

P2 — 3-6 mois

  • Scoring crédit alternatif
  • Escrow automatisé
  • Multi-pays (XOF, GHS, NGN)
  • API B2B MFIs / employeurs

P3 — Licorne (6-18 mois)

  • Licence monnaie électronique
  • 1M+ tx/mois, 10+ pays
  • IA fraude temps réel
  • Crédit garanti épargne/tontine
